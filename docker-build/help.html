Connecting to the environment

You are executing an ephemeral environment that you can SSH into for testing code and mechanisms on an Intel/x86_64 system. Please note that anything not saved in /data will be lost when you exit the environment!

To connect, please SSH to:

     Address: nimbix@%PUBLICADDR%
    Password: %NIMBIXPASSWD%

You will be able to run commands as root using the sudo prefix once in.

You may optionally install an SSH public key for password-less access by clicking on your username in the top right of the portal, then Account Settings, then SSH Keys to manage your SSH keys.
